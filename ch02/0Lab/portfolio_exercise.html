자기소개

안녕하세요! 저는 사용자 중심의 웹 경험을 만드는 것에 열정을 가진 웹 개발자입니다. HTML5, CSS3, JavaScript를 활용한 반응형 웹 디자인을 전문으로 합니다.

저는 지속적인 학습과 새로운 기술 탐구를 통해 항상 성장하고자 노력합니다.

기술 스택

프론트엔드 기술

  • HTML5 - 시맨틱 마크업, 웹 접근성
  • CSS3 - Flexbox, Grid, 애니메이션
  • JavaScript - ES6+, DOM 조작
  • React - 컴포넌트 기반 UI 개발

백엔드 기술

  1. Node.js
  2. Express
  3. MongoDB
